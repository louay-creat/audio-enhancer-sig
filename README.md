\# Audio Enhancer - SIG



Application Python pour enregistrer un audio, réduire le bruit, analyser la qualité du signal et exporter un rapport TXT/PDF.

## 🚀 Quick Start (1 commande)

Sur Windows (PowerShell) :

```powershell
.\setup.ps1

\## Fonctionnalités



\- Enregistrement microphone

\- Réduction du bruit

\- Filtre passe-bande

\- Analyse RMS, SNR, clipping et ZCR

\- Score de qualité audio

\- Affichage waveform et spectrogramme

\- Lecture audio brut et amélioré

\- Export TXT et PDF



\## Installation



```bash

git clone https://github.com/louay-creat/audio-enhancer-sig.git

cd audio-enhancer-sig

python -m venv .venv

.venv\\Scripts\\activate

pip install -r requirements.txt
